Introduction
============

Some organizations, like Greek-letter organizations, assign big brothers or big
sisters ("bigs") to new members ("littles"). This program visualizes such
relationships using Graphviz. It was originally developed for use by the Delta
Alpha Chapter of Sigma Nu Fraternity.
